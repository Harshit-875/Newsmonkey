# NewsMonkey

NewsMonkey is a React-based news application that fetches real-time news from a news API and displays the latest headlines across different categories.

This project was created to learn React development, API integration, routing, and dynamic content rendering.

---

# 📰 Features

* Real-time news fetching using API
* Category-wise news sections
* Infinite scrolling
* Responsive UI
* Dynamic news cards
* Loading spinner
* React Router integration
* Fast and modern interface

---

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Harshit-875/Newsmonkey.git
```

Go to the project directory:

```bash
cd Newsmonkey
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

# 🔑 API Configuration

Create a `.env` file and add your News API key:

```env
VITE_NEWS_API=your_api_key_here
```
